### Installation

[dein.vim](https://github.com/Shougo/dein.vim) / [vim-plug](https://github.com/junegunn/vim-plug)

```vim
call dein#add('haya14busa/vim-stacktrace', {'build': 'make'})
```

```
Plug 'haya14busa/vim-stacktrace', { 'do': 'make' }
```
